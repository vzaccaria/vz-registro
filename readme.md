## Usage

```
Usage:
    vz-registro get -u USER -p PASSWORD -n PERSON
    vz-registro status -u USER -p PASSWORD -r REGNUM
    vz-registro upload -u USER -p PASSWORD -r REGNUM -f DATAFILE
    vz-registro ( -h | --help )

Options:
    -h, --help                       help for vz-registro
    -u, --user USER
    -n, --person PERSON
    -p, --password PASSWORD
    -r, --registro REGNUM
    -f, --file DATAFILE

Arguments:

```
